This app is a task organiser made to help users organise their quests when gaming,

the app allows users to view a summary of quests. Each quest is made up of tasks and,

when the user has completed all the tasks the overall quest is completed.

This app can be used for multiple tasks and is made to suit users' needs and help

them complete their to do list in an organised, aesthetically pleasing enviroment that is also user friendly.
